# 연구과제 예산 계산기

연구과제의 연차별 인건비, 연구수당, 연구활동비, 연구재료비, 간접비, 부가세, 소요 총액과 부족/여유 예산을 계산하는 정적 웹 애플리케이션입니다.

## 주요 기능

- 연차 추가, 복제, 삭제
- 내부/외부/학생 인건비 행 추가, 삭제, 순서 변경
- 내부인건비 계상/미계상 구분
- 직위/과정 선택 시 기본 기준액 자동 입력
- 간접비율, 연구수당율, 부가세 여부, 간접비 계산 기준 설정
- 국가R&D/위탁/용역 간접비 프리셋
- 부족/여유 예산 자동 계산 및 연구활동비로 잔액 맞추기
- 전체 연차 합계 표시
- 엑셀(`.xlsx`) 내보내기

## 로컬 실행

```bash
npm install
npm run dev
```

## 검증

```bash
npm test
npm run build
```

## 배포

`main` 브랜치에 push하면 GitHub Actions가 테스트와 빌드를 수행한 뒤 GitHub Pages에 정적 사이트를 배포합니다. 저장소 Settings → Pages에서 Source가 `GitHub Actions`로 설정되어 있어야 합니다.
